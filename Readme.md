#多个TCP连接传输数据

1、client端发出请求：发送文件｛host、文件名、文件大小｝
2、server端响应client请求：返回server监听端口号｛｝，同时监听相应端口号
3、client将文件切片并发送：接到server返回的端口号后，将切片的文件放入队列，按照端口数量，启动相应数量的文件发送client，并从队列中取出切片数据，发送数据包格式 "I1024s"

#切片策略
1、小于10M不切
2、10M一片,
3、每个切片格式：前2个字节代表切片顺序，后边为数据包
